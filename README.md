## HOW TO RUN
1. python3 설치

```powershell
sudo apt-get update
sudo apt install python3
```

2. 폴더로 이동하여 모듈 설치

```powershell
cd 2022-2-OSSProj-SGC-3
pip install -r requirements.txt
```

3. 게임 실행

```powershell
python3 mainMenu.py
```
